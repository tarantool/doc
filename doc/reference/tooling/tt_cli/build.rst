.. _tt-build:

Build an application
====================

..  code-block:: bash

    tt build PATH [flags]


``tt build `` builds a Tarantool application locally.

Flags
-----

..  container:: table

    ..  list-table::
        :widths: 30 70
        :header-rows: 0

        *   -   ``--spec``
            -   Path to a custom ``.rockspec`` file that you want use for the current build
