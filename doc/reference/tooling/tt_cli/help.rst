.. _tt-help:

Displaying help for tt and its commands
=======================================

..  code-block:: bash

    tt help [COMMAND]

``tt help`` displays help:

*   for ``tt`` utility when called without a ``COMMAND``.
*   for a specified ``tt`` command.

Examples
--------

*   Display ``tt`` help

    ..  code-block:: bash

        tt help

*   Display help for the ``start`` command:

    ..  code-block:: bash

        tt help start
