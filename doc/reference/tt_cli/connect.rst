Connecting to a Tarantool instance
=============================

..  code-block:: bash

    tt connect

Flags
-----

..  container:: table

    ..  list-table::
        :widths: 20 80
        :header-rows: 0

        *   -   ``--name``
            -   Application name.

Details
-------

Your application will appear in the ``<path>/<app-name>/`` directory.
