.. _tt-uninstall:

Uninstalling Tarantool software
===============================

..  code-block:: console

    $ tt uninstall PROGRAM_NAME [VERSION]

``tt uninstall`` uninstalls a previously :doc:`installed <install>` Tarantool version.

Example
--------

Uninstall Tarantool 2.10.4:

..  code-block:: console

    $ tt uninstall tarantool 2.10.4
